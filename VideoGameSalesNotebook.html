It's difficult to compare platforms from different years, specially since our Year ranges from 1980 to 2020, that's 40 years apart! How did the sales changed throughout the years? Naturally, we would expect that the global sales would increase accross the years, and we see that till we reach around 2010, where we see a huge drop. One explanation I can think of is that is coincidenes with the years of the 2009 financial crisis where a lot of people got affected by it.

The other explanation is if we don't have as many games recorded in those years. We can see by the second plot below that we have less unique games for those years after 2010, and that is probably the main reason for the decrese in global sales.

In [18]:
def Platform_to_Owner (x):
    if x in ['PS','PS2','PS3','PS4','PSP','PSV']:
        return 'SONY'
    if x in ['Wii','NES','GB','DS','SNES','GBA','3DS','N64','GC','WiiU']:
        return 'NINTENDO'
    if x in ['X360','XB','XOne']:
        return 'MICROSOFT'
    return 'Other'
